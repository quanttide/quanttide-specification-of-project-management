# 项目模型`Project`

定义“项目集”和“项目组合”为“项目”的子类。
在项目管理理论中，“项目”有狭义和广义两种表述方式，狭义的项目可以是项目集和项目组合的组成部分，广义的项目是他们的总称。
我们这里采纳了广义的表述，并且让项目集和项目组合拥有项目的绝大多数特性，以支持更复杂的项目边界划分策略。

## 标识字段`name`

## 类型字段`type`

选项：

- `default`: 默认
- `program`: 项目集
- `project_portfolio`: 项目组合

## REST API

- list: `/projects`
- detail: `/projects/<name>`
